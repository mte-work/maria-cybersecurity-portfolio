# Linux Fundamentals for Cybersecurity

## _Objective_:
Learn and demonstrate basic Linux skills commonly used in cybersecurity, including
file navigation, permissions, and basic system commands.

## _Why This Matters_:
Linux is widely used in servers, security tools, and cloud environments. Understanding
how to navigate and inspect a Linux system is a foundational cybersecurity skill.

## _Environment_:
- OS: Ubuntu 22.04 (Virtual Machine)
- VM Provider: VirtualBox

## _Summary:_

These skills form the basis for system investigation, access control, and incident response. This section demonstrates foundational Linux skills relevant to cybersecurity, including:
- User identification
- Filesystem navigation
- Permission management
- Awareness of system logs
