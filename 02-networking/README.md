# Networking Fundamentals

## Objective
Understand basic networking concepts and commands used to identify and analyze network connectivity.

## Why This Matters
Cybersecurity depends on understanding how data moves across networks, including IP addressing,
ports, and interfaces.

## Environment
- OS: Ubuntu 24.04 LTS (Virtual Machine)

## Key Skills Demonstrated
- Identifying network interfaces
- Understanding IP addressing at a system level
- Using Linux networking tools for visibility
